# quickjs-cpp
quickjs-cpp is a modern C++ wrapper around the QuickJS JavaScript engine that simplifies embedding and interacting with JavaScript from C++ applications. It provides a type-safe and ergonomic interface for executing scripts, evaluating expressions, and binding native C++ functions and objects to JavaScript.

Built on top of the lightweight and embeddable QuickJS engine, this project enables developers to integrate a fast ES2020+ compliant JavaScript runtime into C++ projects with minimal boilerplate. It abstracts away the low-level C API of QuickJS while preserving performance and flexibility, making it suitable for scripting, extension systems, sandboxing, and embedded runtime environments.

Key features typically include context management, script evaluation, exception handling, and seamless interop between C++ and JavaScript values, helping developers build extensible applications with JavaScript as an embedded scripting layer.
